1. Modification du Prompt de l'IA (Backend/API Route) :
Change le prompt envoyé à OpenAI. Si l'utilisateur sélectionne plusieurs plateformes (ex: Instagram et WhatsApp), demande explicitement à l'IA de renvoyer un objet JSON ou une structure Markdown claire contenant un texte unique et ultra-spécifique pour chaque plateforme cochée.

    Instagram : Texte visuel, axé lifestyle, avec un bloc de hashtags à la fin.

    WhatsApp : Texte direct, aéré, structuré avec des listes à puces, et un appel à l'action clair vers le catalogue ou une discussion.

    TikTok/Snapchat : Une accroche textuelle ultra-courte ou un mini-script vidéo dynamique.

2. Modification de l'Affichage (Frontend) :
Dans la section 'COPIE GÉNÉRÉE', ne montre plus un seul bloc brut. Si plusieurs plateformes ont été cochées, utilise un système d'onglets cliquables (Tabs) ou une liste de sections bien séparées (ex: 'Pour Instagram', 'Pour WhatsApp') avec un bouton 'Copier' indépendant pour chaque plateforme.

3. Déploiement :
Vérifie la validité du code avec npm run build puis déploie la mise à jour sur Vercel avec npx vercel --prod."

"Modifie l'intégration de PayDunya dans l'application pour utiliser l'API de paiement direct (Checkout Invoice) à la place du mode de connexion par compte.

Objectif : L'utilisateur final ne doit jamais avoir à créer de compte, se connecter ou saisir un mot de passe sur PayDunya. Le parcours doit être ultra-rapide.

Logique à implémenter :

    Interface utilisateur (src/app/page.tsx) : >    - Lorsque les 5 essais gratuits sont épuisés, affiche un formulaire simple avec :

        Un champ pour saisir le Numéro de téléphone (ex: 77XXXXXXX / 76XXXXXXX).

        Le montant fixe affiché : 500 FCFA.

        Le bouton violet 'Payer & Générer'.

    Logique Backend (API Route PayDunya) :

        Configure l'appel API vers PayDunya en créant une facture directe (Paydunya::Checkout::Invoice).

        Passe le montant à 500 et ajoute le numéro de téléphone de l'utilisateur dans les métadonnées de la facture de sorte que lorsqu'il arrive sur la page de paiement, PayDunya pré-remplisse le champ Wave/Orange Money ou lance directement la demande de paiement sans demander d'e-mail ou de mot de passe.

        Utilise le système de redirection automatique après succès vers notre URL de retour pour délivrer les copies d'onglets (WhatsApp, IG, TikTok).

Vérifie la compilation avec npm run build et déploie le correctif propre sur Vercel avec npx vercel --prod."
